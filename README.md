# ESP8266_ALEXA_01REL-
Este projeto consiste em um sistema de automação residencial utilizando um microcontrolador baseado no ESP8266, permitindo o controle de um relé através de comandos de voz com assistentes virtuais compatíveis. A comunicação é realizada via rede Wi-Fi, possibilitando integrar dispositivos físicos a um ambiente de Internet das Coisas (IoT).
